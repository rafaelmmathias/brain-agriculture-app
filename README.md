# Getting Started

You just need create a `.env` file in the root folder with the following API endpoint.

Example:
```
REACT_APP_API_ENDPOINT=http://localhost:4000
```

Running the app:
```
yarn install

yarn start
```

# Unit tests

### `yarn tdd`
Runs tests in watch mode for development time

### `yarn test`
Runs tests withou watch (for CI)
